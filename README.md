# Gradio-приложение: Линейная регрессия

**Цель:** продемонстрировать использование Gradio для простой веб-формы, предсказания линейной модели, вывода R² и визуализации обучающих точек + введённой пользователем точки.
https://github.com/VetrovSV/Programming/blob/master/plans/ML/tasks/gradio.md

## Структура проекта
gradio/

├─ app/

│  └─ app.py                # Gradioприложение

├─ models/

│  └─ model.joblib          # ваша сохранённая модель (скопируйте сюда)

├─ requirements.txt         # минимальные зависимости для запуска (pip)

├─ modrequirements.txt      # перечисление всех библиотек, используемых в коде 

├─ README.md                # инструкция

└─ tests/

   ├─ test_predict.py       # (бонус) тест для предсказания
   
   └─ models/
   
	└─ model.joblib 


## Что делает приложение
1. Загружает предварительно обученную модель `models/model.joblib`.
2. Генерирует синтетические данные
    Принимает на вход 4 признака (x1..x4) через Gradio-форму и возвращает:
  - Предсказанное значение y.
  - Scatter-диаграмму: обучающие точки + ваша введённая точка (выделена).
  - В `description` отображается R² на тестовой выборке и уравнение регрессии в LaTeX (коэффициенты округлены до 2 знаков).
3. На графике дополнительно показана линия регрессии вдоль наиболее важного признака (остальные признаки фиксированы на средних значениях).Шаги по установке

## Шаги по установке

1.  ### Создание папки проекта и виртуального окружения в cmd
```
mkdir gradio_project
cd gradio_project
```

2.	### Создайте и активируйте окружение
	
	```
	python -m venv .venv
	.venv\Scripts\activate.bat
	```
	
3.	### Установите зависимости
```
pip install -r requirements.txt
pip install -r modrequirements_service.txt
```
Проверка установленных пакетов
```
pip list
```

4.	### Загрузите модель ***
	```
	python lin_reg.py
	```
	
	
	
5.	### Запустите app.py и перейдите в браузере http://127.0.0.1:7860/
	```
	python app.py
	```
